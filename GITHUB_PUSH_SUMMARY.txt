HOW TO PUSH TO GITHUB
================================================================================

You need to authenticate with GitHub. Choose ONE method:

OPTION 1: GitHub CLI (Recommended - Easiest)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Install GitHub CLI:
   brew install gh           # macOS
   sudo apt-get install gh   # Ubuntu/Debian

2. Authenticate:
   gh auth login
   # Follow browser prompts

3. Push:
   cd /mnt/user-data/outputs
   git push -u origin main

4. Verify:
   https://github.com/shresth111/cloud-guest


OPTION 2: Personal Access Token (GitHub UI)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Generate token: https://github.com/settings/tokens
   - Click "Generate new token (classic)"
   - Select "repo" scope
   - Copy the token

2. Push:
   cd /mnt/user-data/outputs
   git push -u origin main
   # Paste token when prompted for password

3. Verify:
   https://github.com/shresth111/cloud-guest


OPTION 3: SSH Key
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Generate SSH key:
   ssh-keygen -t ed25519 -C "shresth.pathak@verbaflo.ai"
   cat ~/.ssh/id_ed25519.pub

2. Add to GitHub: https://github.com/settings/keys

3. Update remote:
   cd /mnt/user-data/outputs
   git remote set-url origin git@github.com:shresth111/cloud-guest.git

4. Push:
   git push -u origin main
